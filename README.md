# Lesspay2 API Multi-language Demos

This repository contains official API integration demos for Lesspay2.

## Structure

- [**Java Demo**](./lesspay2-api-demo-java/README.md) - Spring Boot based example with complete signature verification and API calls.

## Common Concepts

All SDKs implement the SHA256 signature algorithm required by Lesspay2. Please refer to the specific language directory for implementation details.
